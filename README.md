# blescan-rpi
----
Note: It been tested on Raspberry Pi 3 Module B with Raspbian OS.
It is a BLE scanning program for your Raspberry Pi 3 Module B, based on C language. 
----
* How to install.
* git clone https://github.com/yoyojacky/blescan-rpi.git 
* cd blescan-rpi/
* chmod +x install.sh 
* bash install.sh 
* ./blescan
----
